# 기업 월별 진행 리포트 PRD

## 배경

현재 사이트는 문의 접수, 자료 첨부, 관리자 확인 기능을 Supabase 기반으로 제공한다. 다음 단계로 계약/컨설팅이 진행 중인 기업이 월별 현황을 확인하고 출력할 수 있는 운영 리포트 기능이 필요하다.

## 목표

- 관리자 작성/편집, 초안 저장, 공개 전환 기능을 제공한다.
- 기업 로그인 사용자는 자기 기업의 공개 리포트만 볼 수 있다.
- 월별 컨설팅 내용, 공고, 사업계획서, 미팅, 파일을 한 화면에 모아 출력할 수 있다.

## 범위

- 포함: Supabase 스키마, RLS, 관리자 리포트 작성 UI, 기업 로그인 연결, 기업 포털, 인쇄 CSS.
- 제외: 서버 PDF 생성, 자동 AI 리포트 생성, 기업 파일 업로드, 이메일/카카오 알림 자동 발송.

## 기능 요구사항

- 관리자는 기업명과 기업 담당자 이메일을 기준으로 기업 정보를 관리할 수 있다.
- 관리자는 월별 리포트를 생성하고 `draft` 또는 `published` 상태로 저장할 수 있다.
- 관리자는 공고 항목을 최대 5개까지 입력할 수 있다.
- 관리자는 사업계획서 컨설팅 항목을 최대 3개까지 입력할 수 있다.
- 관리자는 미팅 항목을 최대 3개까지 입력할 수 있다.
- 관리자는 리포트 파일을 업로드하고 리포트와 연결할 수 있다.
- 기업 담당자는 로그인 후 공개된 리포트 목록과 상세 내용을 조회할 수 있다.
- 리포트 상세 화면에는 인쇄 버튼이 있고, 인쇄 시 내비게이션과 버튼이 제외된다.

## 비기능 요구사항

- Supabase 설정값이 비어 있으면 명확한 안내를 보여준다.
- 기존 문의 접수, 관리자 문의 목록, 첨부파일 열람 기능은 유지한다.
- 모바일 화면에서도 표와 리포트 내용이 가로 깨짐 없이 읽히도록 한다.
- 파일 열람은 signed URL을 사용한다.

## 성공 기준

- 관리자 계정으로 로그인해 리포트를 저장하고 공개할 수 있다.
- 기업 계정으로 로그인하면 자기 회사의 공개 리포트만 표시된다.
- 브라우저 인쇄 미리보기에서 리포트 본문만 출력된다.
- Supabase RLS로 다른 회사 리포트와 초안 리포트 조회가 차단된다.
