# 기업 월별 진행 리포트 기획서

## 목적

관리자와 기업 담당자가 같은 사이트 로그인 체계 안에서 기업별 월간 컨설팅 진행 상황을 확인하고, 월별 리포트를 브라우저 인쇄 또는 PDF 저장으로 출력할 수 있게 한다.

## 사용자

- 관리자: 기업, 월별 리포트, 공고, 사업계획서, 미팅, 파일을 작성하고 공개한다.
- 기업 담당자: 자기 기업에 공개된 월별 리포트를 조회하고 출력한다.

## 핵심 흐름

1. 관리자가 기업을 등록하거나 기존 기업을 선택한다.
2. 관리자가 리포트 월, 제목, 기획 요약, 컨설팅 내용을 입력한다.
3. 관리자가 공고 맞춤 리스트와 접수 상황을 입력한다.
4. 관리자가 사업계획서 작성 수준과 컨설팅 내용을 입력한다.
5. 관리자가 미팅 내용과 녹음/첨부 파일을 등록한다.
6. 관리자가 리포트를 초안으로 저장하거나 공개한다.
7. 기업 담당자는 기업 로그인 후 공개 리포트만 조회한다.
8. 기업 담당자 또는 관리자는 리포트 상세 화면에서 인쇄/PDF 저장을 실행한다.

## 리포트 구성

- 기본 정보: 기업명, 리포트 월, 작성일, 공개 상태, 담당자
- 기획 요약: 이번 달 목표, 핵심 진단, 다음 액션
- 공고: 맞춤 공고명, 기관, 마감일, 적합도, 접수 상태, 보완 필요 사항
- 사업계획서: 작성 수준, 컨설팅 내용, 다음 보완 항목
- 미팅 관리: 미팅일, 제목, 참석자, 회의 내용, 후속 과제
- 파일: 미팅 녹음, 첨부 파일, 기타 검토 자료

## 운영 원칙

- 기업 공개는 `draft` 상태에서는 차단하고 `published` 상태에서만 허용한다.
- 파일 업로드는 관리자만 수행한다.
- 기업 사용자는 자기 회사에 매핑된 공개 리포트와 파일만 조회한다.
- 출력은 별도 PDF 서버 없이 인쇄 전용 CSS로 제공한다.
